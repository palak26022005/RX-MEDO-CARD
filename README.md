# RX-MEDO-CARD

RX-MEDO-CARD is a modern healthcare web application designed to simplify medical record management and improve the interaction between patients, doctors, and healthcare providers. The platform provides a secure and user-friendly digital solution for storing, accessing, and managing essential medical information.

## Features

* Secure digital storage of medical records
* Easy access to patient health information
* User-friendly and responsive interface
* Fast and efficient record retrieval
* Doctor and patient information management
* Streamlined healthcare data organization

## Technologies Used

* HTML5
* CSS3
* JavaScript
* PHP
* MySQL
* XAMPP

## Project Objectives

* Digitize and centralize medical records
* Improve accessibility of healthcare information
* Reduce paperwork and manual record handling
* Enhance efficiency in healthcare management
* Provide a reliable and secure medical data platform

## Installation and Setup

1. Clone the repository:

   ```bash
   git clone https://github.com/palak26022005/RX-MEDO-CARD.git
   ```

2. Move the project folder to your XAMPP `htdocs` directory.

3. Start Apache and MySQL from the XAMPP Control Panel.

4. Import the database into phpMyAdmin.

5. Open your browser and run:

   ```
   http://localhost/RX-MEDO-CARD/
   ```
